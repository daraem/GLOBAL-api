<h1>Endpoints</h1>

- fullschedule: gets entire GLOBAL busses schedules
- getschedule: takes 1 parameter *id*, returns entire schedule of *id*'s bus line
- getschedule_ida: takes 1 parameter *id*, returns coming schedule of *id*'s bus line
- getschedule_vuelta: takes 1 parameter *id*, returns returning schedule of *id*'s buss line
